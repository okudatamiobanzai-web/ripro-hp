# site/candidates/

HP 画像差し替え用の候補素材置き場。`manifest.json` が各ファイルの内容と割り当て先スロット候補を持つ。

## 運用

- 画像選定ツール（ピッカー）が `manifest.json` を読み込んで表示する想定。
- 現状は **元解像度のまま** コピーされている。`process_images.sh` を一度実行して `sips` で max 1200px / JPEG q85 に圧縮する（サンドボックス制約で初回コピー時に sips を走らせられなかった）。
- `ロゴ_01.png` のみ透過維持のため PNG のまま据え置き。

## ファイル数

25 枚（品目別24 + ロゴ1） + 現場写真 1（廃プラ堆積_01.jpeg）= 合計 26 枚。
